# Orpington News

Orpington News is a self-hosted RSS/Atom feed reader with PWA support.

## Features

- Supports RSS 1.0/RSS 2.0/Atom feeds
- Supports nested collections (groups of feeds)
- Light and dark theme
- Syntax highlighting
- Mobile-friendly UI
- PWA support
